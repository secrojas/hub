# Milestone v1.0: MVP

**Status:** ✅ SHIPPED 2026-04-01
**Phases:** 1–7
**Total Plans:** 20

## Overview

Hub se construye en siete fases ordenadas por dependencias duras: la autenticación y los roles de acceso abren el camino; el CRUD de clientes habilita todos los módulos de datos; tareas/Kanban entrega el valor central; facturación y presupuestos completan la gestión operativa; el portal del cliente agrega todos los módulos en una vista de solo lectura; y el dashboard del admin corona la plataforma con visibilidad inmediata de lo que está activo y lo que vence pronto.

## Milestone Summary

**Key Decisions:**

- Laravel 12 used (plan specified 11.x) — compatible upgrade, all patterns identical
- Auth: Breeze Vue+Inertia CSR preset, register routes disabled (invitation-only)
- PDF: barryvdh/laravel-dompdf v3.1.2, streamDownload, DejaVu Sans para UTF-8/ARS
- Portal: lista simple de tareas (no Kanban interactivo para el cliente)
- Kanban: vue-draggable-plus uses @add + event.data (NOT @change + event.added.element)
- Two-query dashboard: vencenProonto first (7-day window), enProgreso excludes those IDs
- nullOnDelete on billings.client_id and quotes.client_id — financial records survive client deletion

**Issues Resolved:**

- v-for on objects: Vue 3 uses (value, key) order — fixed Kanban column label bug
- vue-draggable-plus API mismatch: @change never fires for cross-list drag — switched to @add
- dompdf default font drops ñ/á/é silently — fixed with DejaVu Sans

**Issues Deferred:**

- Notificaciones automáticas al cliente → v2
- Múltiples admins / modo equipo → v2
- Historial de presupuestos filtrable → v2

**Technical Debt:**

- Phase 6 Portal: deferred-items.md lists PDF download UI improvements and empty states polish
