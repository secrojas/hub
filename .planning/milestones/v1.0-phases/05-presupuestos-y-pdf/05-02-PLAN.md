1. Create `app/Http/Requests/StoreQuoteRequest.php`:
   ```php
   public function rules(): array
   {
       return [
           'client_id' => ['required', 'exists:clients,id'],
           'titulo'    => ['required', 'string', 'max:255'],
           'notas'     => ['nullable', 'string'],
           'items'     => ['required', 'array', 'min:1'],
           'items.*.descripcion' => ['required', 'string', 'max:500'],
           'items.*.precio'      => ['required', 'numeric', 'min:0.01'],
       ];
   }
   ```

2. Create `app/Http/Requests/UpdateQuoteRequest.php`:
   Same rules as StoreQuoteRequest. The controller will check estado === borrador before allowing content updates.

3. Create `app/Http/Controllers/QuoteController.php` with these methods:

   **index(Request $request):**
   - Query `Quote::with('client', 'items')->latest()`
   - No filters needed (QUOT-04 filterable history is v2 / deferred)
   - Compute a `total` attribute by summing items on each quote for display: use `->get()->map(fn ($q) => [...$q->toArray(), 'total' => $q->items->sum('precio')])` — or paginate(20) and append total via `->through(fn ($q) => [... , 'total' => $q->items->sum('precio')])`
   - Pass clients list for reference: `Client::orderBy('nombre')->get(['id', 'nombre'])`
   - Render `Admin/Quotes/Index`

   **create():**
   - Pass `clients` list: `Client::orderBy('nombre')->get(['id', 'nombre'])`
   - Render `Admin/Quotes/Create`

   **store(StoreQuoteRequest $request):**
   - `$quote = Quote::create($request->safe()->except('items'))`
   - Loop `$request->validated()['items']` → `$quote->items()->create($item)`
   - Redirect to `quotes.index`

   **edit(Quote $quote):**
   - `$quote->load(['client', 'items'])`
   - Pass `clients` list
   - Render `Admin/Quotes/Edit`

   **update(UpdateQuoteRequest $request, Quote $quote):**
   - Guard: `abort_if($quote->estado !== QuoteStatus::Borrador, 403)` — content edit blocked post-borrador
   - `$quote->update($request->safe()->except('items'))`
   - `$quote->items()->delete()` then re-insert all items from `$request->validated()['items']`
   - Redirect to `quotes.index`

   **destroy(Quote $quote):**
   - Guard: `abort_if($quote->estado !== QuoteStatus::Borrador, 403)` — delete only in borrador
   - `$quote->delete()` (cascade deletes items)
   - Redirect to `quotes.index`

   **updateEstado(Request $request, Quote $quote):**
   - Validate: `['estado' => ['required', Rule::enum(QuoteStatus::class)]]`
   - `$quote->update(['estado' => $request->estado])`
   - `return back()`
