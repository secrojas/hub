    **Step 1 — Create `resources/js/Pages/Admin/Knowledge/Index.vue`:**

    Structure:
    ```vue
    <script setup>
    import { ref, watch } from 'vue'
    import { Link, router } from '@inertiajs/vue3'
    import AdminLayout from '@/Layouts/AdminLayout.vue'

    defineOptions({ layout: AdminLayout })

    const props = defineProps({
        entries:     Array,
        filters:     Object,
        types:       Array,   // [{label, value}]
        statuses:    Array,   // [{label, value}]
    })

    const search = ref(props.filters?.search ?? '')
    const type   = ref(props.filters?.type ?? '')
    const status = ref(props.filters?.status ?? '')
    const domain = ref(props.filters?.domain ?? '')

    let debounce = null
    function applyFilters() {
        clearTimeout(debounce)
        debounce = setTimeout(() => {
            router.get(route('knowledge.index'), {
                search: search.value || undefined,
                type:   type.value || undefined,
                status: status.value || undefined,
                domain: domain.value || undefined,
            }, { preserveState: true, preserveScroll: true, replace: true })
        }, 300)
    }
    watch([search, type, status, domain], applyFilters)
    </script>
    ```

    Template layout:
    - Header row with "Knowledge Base" title and `<Link :href="route('knowledge.create')">` button (styled as primary button: `bg-violet-600 hover:bg-violet-700 text-white px-4 py-2 rounded-lg text-sm font-medium`)
    - Filter bar: text input for search, `<select>` for type (with empty "Todos" option + options from `types` prop), `<select>` for status (same pattern), text input for domain
    - Entries table or card list. For each entry show: `entry_id`, `titulo`, `type` (as Badge), `status` (as Badge), `domain`, `updated_at`
    - Each row links to `route('knowledge.show', entry.id)`
    - Empty state: "No hay entradas de conocimiento" message when entries array is empty
    - Use Tailwind classes consistent with existing admin pages (dark theme: bg-surface-900, text-slate-100, etc.)
